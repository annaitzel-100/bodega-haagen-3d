# bodega-haagen-3d
Modelo 3D interactivo optimizado para dispositivos móviles que representa un centro de distribución en cadena de frío para productos Häagen-Dazs. Incluye estructura industrial, áreas operativas delimitadas, flujo logístico unidireccional y sistema de refrigeración a -25°C, desarrollado como proyecto académico de logística y diseño.…………….. …………………..
